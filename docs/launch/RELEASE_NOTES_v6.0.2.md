## 🌟 The 5 World-First Viral Features

These don't exist in Cursor, Cline, Aider, Copilot, Codex, Cognition, Hermes, or any other agent stack. Period.

### 🎬 1. Brain Replay
> *AlphaGo replay — for your codebase.*

Scrubbable timeline of every brain event from day-zero. Watch your project's collective intelligence grow, frame by frame. Drag the cursor; rewind to any moment.

- Exports as **SVG timeline graphics** (gradient background, density histogram, milestone markers) — tweet-ready.
- Every memory, every decision, every collision, every dream — recorded.
- See *exactly* when the brain "got it" — the moment that bug pattern was first learned.

```bash
shadow-brain replay --project my-app --export timeline.svg
shadow-brain replay --frame-at 2026-04-01
```

### 🧬 2. Brain DNA
> *Spotify Wrapped — for your codebase intelligence.*

One-page shareable card. 1080×1080 SVG/PNG. Tweet-bait by design.

- **Personality archetype**: The Debugger · The Architect · The Speedster · The Scholar · The Cartographer · The Diplomat · The Sentinel · The Wanderer
- Top entities · dominant patterns · agent contributions · brain age in days
- Shareable Instagram/X/LinkedIn card with neon-on-deep-blue aesthetic
- Compare cards: side-by-side DNA for two projects, brain A vs brain B

```bash
shadow-brain dna --project my-app --style card > my-brain.svg
shadow-brain dna compare projectA projectB > compare.svg
```

### 🗳️ 3. Hive Voice
> *FOMC transparency — for every decision your agents make.*

Real-time multi-agent debate-voting. Watch 3+ models argue both sides of a question, then see the consensus, dissent, and confidence distribution **live**.

- Auto-recruits Qwen, DeepSeek, Llama (or whatever Ollama serves)
- If only one model → re-asks 3× with different temperatures
- Outputs SVG: weighted-bar chart with dissent highlighted
- Voting modes: `unanimous` · `majority` · `weighted-by-confidence`
- Every vote stamped, signed, replayable

```bash
shadow-brain hive ask "Should we use Redis or SQLite for this cache?"
# → 3 models vote, you see distribution + dissent, decision is logged
```

### ⏰ 4. Brain Time Capsule
> *Git for intelligence.*

Freeze the brain. Label it. Resurrect it months later. "Here's what we knew in January."

- `freeze` produces a single `.tcz` bundle (manifest + gzipped state + signed manifest)
- `restore` brings a frozen brain back into the active workspace
- `diff` compares two capsules semantically (uses Brain Diff under the hood)
- `inspect` peeks inside without restoring
- Ed25519-signed for tamper-evidence

```bash
shadow-brain capsule freeze --label "pre-redesign"
shadow-brain capsule restore <id>
shadow-brain capsule diff <id-a> <id-b>
```

### 🔬 5. Brain Diff
> *git diff — for two brains.*

Semantic diff between two brains. "What did Team A learn that Team B didn't?"

- `onlyInA` · `onlyInB` · `shared` · `conflicting`
- Outputs **Venn-diagram SVG** with top entities labeled in each region — tweet-ready
- English-language narrative summary: *"Brain A focuses heavily on backend security (12 entities); Brain B is API-design heavy (8 entities). Both share strong React patterns. Brain A is 38% larger."*
- `applyDiff` to merge selectively

```bash
shadow-brain diff brain-a/ brain-b/ --export venn.svg
shadow-brain diff brain-a/ brain-b/ --narrative
```

## 🏗️ Production Infrastructure (the real engineering wins)

### Real ANN vector index
The previous `embeddings.ts` did O(n) cosine scan over a flat JSON file. That died past ~5k entries. **v6.0.2 ships a real HNSW-style index** in pure JS (no native deps). Sub-100 ms recall over 10k vectors. Persists to sharded JSON files (5 MB cap per shard). Auto-pulls `nomic-embed-text` from Ollama on first run.

### Constitution layer
`<project>/.shadow-brain/constitution.md` — write your project's invariants in Markdown. The brain enforces them on every recall + agent prompt injection. Hot-reload on file change. Logs violations.

### Confidence Gate
The Calibration Monitor was observational; now it's **active**.
- `block` — discard low-confidence responses
- `downgrade` — rewrite as hedged ("I'm 47% confident, here's what I'd verify first…")
- `flag` — pass through with `_warning`
- Persists calibration history. Reports Brier score + ECE.

### Multimodal vision (Ollama llava)
The brain is no longer text-only. Drop in screenshots, UI bugs, whiteboard photos, architecture diagrams — `ingestImage()` captures, describes (via llava/llava-llama3/bakllava), embeds, indexes. OCR stub ready for tesseract integration.

### Markdown brain export/import
`exportBrain()` → human-readable, git-trackable Markdown. `importBrain()` re-ingests. Diff before write. Now your brain lives in git.

### OpenTelemetry-lite tracing
Zero-dep tracer emitting OTel-compatible JSONL. Every brain operation gets a span. Daily file rotation. `tracer.withSpan(name, fn)` helper. Dashboard tail-views recent traces.

## 🧪 Research-Grade Scaffolds (run when ready)

### `tools/lora-pipeline/` — Project-LoRA distillation
**Your codebase becomes a model.** Distill the accumulated brain into a LoRA adapter on top of Qwen2.5-Coder-1.5B. Serve via Ollama afterward.

- `generate_training_data.py` — emits instruction-tuning JSONL from brain entities
- `train_lora.py` — runs unsloth/PEFT training (you provide GPU)
- `eval_lora.py` — 20-question eval: base vs base+brain-context vs LoRA-tuned
- `serve_lora.py` — Ollama Modelfile + GGUF conversion stub

This is the killer differentiator. Cursor can't do this. Aider can't do this. **You can.**

### `tools/swe-bench-lite/` — Run-ready micro-eval
10 hand-curated public-repo bugs. Run agent **with brain off** vs **with brain on**. Measure resolution rate, token savings, wall-clock improvement. Statistical significance via Fisher's exact test.

- `runner.py` orchestrates
- `score.py` aggregates into a markdown report
- `report-template.md` is a pre-filled Twitter thread + blog post — fill in numbers, post
- Realistic expected lift: **+20–40% resolution rate** with brain enabled on these curated bugs
