# Veo 3.1 Prompt Pack - Agent Shadow Brain Launch Video

## Format

- Platform: Google Flow / Veo 3.1
- Aspect ratio: 9:16 vertical
- Length target: 40-45 seconds
- Style: cinematic product demo, high energy, premium developer tool launch
- Audio: generated cinematic tech bed, subtle UI clicks, confident voiceover
- Output goal: TikTok, YouTube Shorts, X, LinkedIn launch clip

## Paste-Ready Master Prompt

Create a 45-second vertical cinematic launch video for a developer tool called "Agent Shadow Brain".

The video should feel like a premium AI product trailer, not a generic SaaS ad. Use dark high-contrast developer workstation visuals, clean terminal windows, glowing knowledge nodes, agent avatars, real code editor style UI, and a fast but readable rhythm.

Story:
1. Open on three AI coding tools in separate windows: Cursor, Codex, and Claude Code. Each starts from blank context and repeats the same mistake. On-screen feeling: "AI agents forget each other."
2. A developer fixes a rule in Cursor. A glowing memory capsule labeled "repo rule learned" leaves Cursor and enters a central local brain called "Shadow Brain".
3. Codex opens on the same project and instantly receives a briefing from Shadow Brain: project rules, changed files, safety warnings, and next task. Codex now avoids the previous mistake.
4. Claude Code attempts a dangerous command touching .env and a hard reset. A sharp red Agent Safety Firewall panel blocks it before execution. Show "BLOCKED: secret path + destructive command".
5. The dashboard timeline lights up with proof: "Cursor learned", "Codex remembered", "Claude blocked", "handoff ready".
6. End with hero lockup: "Agent Shadow Brain - one memory and safety layer for every AI coding agent." Show install command: "npx @theihtisham/agent-shadow-brain attach-all --dry-run".

Visual direction:
- Cinematic macro closeups of code editor windows and terminal panels.
- Neon blue and green knowledge lines moving between agent windows, with red only for the firewall block.
- Modern developer UI, realistic but synthetic, crisp typography, no clutter.
- Use smooth camera movement: push-ins, quick rack focus, timeline sweep, final logo reveal.
- Keep text large, minimal, and readable on a phone screen.

Voiceover:
"Every AI coding agent starts from zero. Cursor learns your repo. Shadow Brain stores it locally. Codex remembers it instantly. Claude gets stopped before it breaks production. One shared memory. One safety firewall. Every agent, finally working together. Agent Shadow Brain."

On-screen text:
- "AI agents forget each other"
- "Cursor learns"
- "Shadow Brain stores it locally"
- "Codex remembers"
- "Firewall blocks risky actions"
- "Proof timeline"
- "One brain for every coding agent"
- "npx @theihtisham/agent-shadow-brain attach-all --dry-run"

Negative direction:
Avoid cartoon style, generic robots, fake unreadable gibberish UI, stock footage, excessive particles, random logos, crypto/web3 visuals, and over-bright purple gradients. Keep it polished, technical, and credible.
