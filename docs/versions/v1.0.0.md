# v1.x — Foundation

> Released: 2026-02 to 2026-03 | [GitHub Release](https://github.com/theihtisham/agent-shadow-brain/releases/tag/v1.0.0)

## Highlights

The **foundation release** — Agent Shadow Brain starts as a universal AI coding intelligence layer that works with any AI coding agent. Health scoring, smart fixes, vulnerability scanning, and CI/CD integration out of the box.

### Added

## v1.0.0 — Initial Release

### Features
- **7 agent adapters** — Claude Code, Cursor, Cline, Windsurf, Copilot, Kilo Code, Amp
- **Health scoring** — overall project health with weighted metrics
- **Analysis pipeline** — extensible code analysis framework
- **Vulnerability scanning** — basic security pattern detection
- **CLI** — 15 commands for project analysis and management
- **Node.js >= 18** — modern ESM architecture
- **TypeScript** — fully typed codebase with strict mode

### Supported Agents
| Agent | Injection Target |
|---|---|
| Claude Code | `CLAUDE.md`, `.claude/` |
| Cursor | `.cursor/mcp.json` |
| Cline | `.clinerules` |
| Windsurf | `.windsurfrules` |
| GitHub Copilot | `.github/copilot-instructions.md` |
| Kilo Code | `.kilocode/rules.md` |
| Amp | `.amp/rules.md` |
