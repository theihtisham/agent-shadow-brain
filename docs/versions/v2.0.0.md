# v2.0.0 — Super-Intelligence Edition

> Released: 2026-03-18 | [GitHub Release](https://github.com/theihtisham/agent-shadow-brain/releases/tag/v2.0.0)

## Highlights

v2.0.0 is a **complete architecture rewrite** — the Shadow Brain transforms from a simple analysis tool into a full super-intelligence platform with semantic understanding, dependency analysis, code similarity detection, MCP protocol support, and team collaboration.

---

## New Features

### Semantic Analyzer
- Deep code understanding beyond syntax parsing
- Intent recognition for code blocks
- Semantic relationship extraction between code entities
- Natural language summaries of complex logic

### Dependency Graph
- Full project dependency visualization
- Circular dependency detection
- Unused dependency identification
- Upgrade impact analysis

### Code Similarity Engine
- Detect duplicated and near-duplicated code across the project
- Cross-file similarity scoring
- Refactoring suggestions for common patterns
- Clone detection with configurable thresholds

### MCP Server (Initial)
- Model Context Protocol server for AI tool integration
- Initial tool set for health check and basic analysis
- Foundation for the full 19-tool server in v5.0.1

### Team Mode
- Collaborative AI-assisted development
- Shared insight pools across team members
- Conflict-free knowledge merging

---

## Breaking Changes from v1.x

- CLI command structure reorganized — see `shadow-brain --help` for new layout
- Configuration file format changed (auto-migrated on first run)
- Minimum Node.js version raised to 18

---

## Upgrade from v1.x

```bash
npm install -g @theihtisham/agent-shadow-brain@2.0.0
```

Configuration is automatically migrated on first run.
