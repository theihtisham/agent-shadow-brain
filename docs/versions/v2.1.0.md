# v2.1.0 — Quantum Neural Mesh

> Released: 2026-03-25 | [GitHub Release](https://github.com/theihtisham/agent-shadow-brain/releases/tag/v2.1.0)

## Highlights

v2.1.0 introduces the **Neural Mesh** — a cross-session shared intelligence layer powered by information theory. Shannon entropy, cosine similarity, and Bayesian confidence scoring bring mathematical rigor to code analysis.

---

## New Features

### Cross-Session Shared Intelligence
- Insights persist across sessions and agent restarts
- Knowledge accumulates over time, forming long-term project memory
- Shared across all connected AI agents

### Shannon Entropy Analysis
- Information-theoretic measurement of code complexity
- High entropy = high information density (complex logic)
- Low entropy = repetitive/boilerplate code
- Used to prioritize analysis effort

### Cosine Similarity Matching
- Semantic code comparison using vector embeddings
- Detects structurally similar code even with different variable names
- Powers deduplication and pattern recognition

### Bayesian Confidence Scoring
- Every insight includes a probabilistic confidence score
- Prior beliefs updated with new evidence
- Confidence grows with corroborating observations
- Low-confidence insights flagged for human review
