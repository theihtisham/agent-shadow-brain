## Highlights

v3.0.0 adds **cognitive intelligence** — the Shadow Brain now understands mental complexity, security risks, and code influence patterns. Three new analysis engines bring human-like reasoning to code review.

---

## New Features

### Cognitive Load Analysis
Estimates the mental effort required to understand code sections:
- **Intrinsic load** — inherent complexity of the logic
- **Extraneous load** — unnecessary complexity from poor structure
- **Germane load** — complexity that aids understanding (good abstractions)
- Per-function and per-file cognitive scores
- Recommendations to reduce cognitive overhead

### Security Audit Engine
Automated vulnerability scanning:
- **Dependency risk assessment** — known CVEs, outdated packages
- **Code pattern scanning** — SQL injection, XSS, command injection, path traversal
- **Secret detection** — API keys, passwords, tokens in source code
- **Configuration audit** — insecure defaults, missing headers
- Severity scoring: critical, high, medium, low, info

### Influence Map
Tracks how code changes ripple through the codebase:
- **Change propagation** — which files are affected when you change X
- **Coupling metrics** — tight vs loose coupling detection
- **Hotspot identification** — files that trigger the most downstream changes
- **Impact scoring** — quantified influence per file/function
