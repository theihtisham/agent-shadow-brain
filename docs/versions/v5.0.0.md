## New Features

### Hierarchical Memory Compression
4-tier memory pyramid that compresses knowledge while preserving access:

```
     /\
    /  \     Principle (highest compression, most general)
   /----\
  / Pat. \   Pattern (recurring structures, templates)
 /--------\
/ Summary  \ Summary (condensed versions of raw data)
/============\
|    Raw     | Raw (full detail, recent data)
```

- Automatic promotion from raw -> summary -> pattern -> principle
- Drill-down: navigate from principle back to supporting raw data
- Drill-up: see how raw observations form patterns and principles
- Configurable retention at each tier

### Context-Triggered Associative Recall
Memory activates automatically based on what you're working on:

| Trigger | Example |
|---|---|
| File path | Opening `auth.ts` recalls all auth-related memories |
| Keywords | Error message keywords trigger past solutions |
| Category | Security work recalls security patterns |
| Temporal | Recent patterns surface first |

### Multi-Agent Consensus Protocol
When multiple AI agents analyze the same code, consensus is built through:

- **Trust-weighted voting** — agents with better track records have more influence
- **Confidence intervals** — insights include uncertainty bounds
- **Conflict resolution** — automated reconciliation of contradictory insights
- **History tracking** — consensus accuracy measured over time

### Collective Cross-Project Learning
Knowledge transfers between projects:

- **Verified rules** — only proven patterns propagate
- **Viral propagation** — high-accuracy rules spread automatically
- **Accuracy tracking** — rules lose trust if they fail in new contexts
- **Opt-in sharing** — project owners control what gets shared
