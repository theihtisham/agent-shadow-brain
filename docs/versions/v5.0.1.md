# v5.0.1 — Zero-Config Intelligence Edition

> Released: 2026-04-15 | [npm](https://www.npmjs.com/package/@theihtisham/agent-shadow-brain/v/5.0.1) | [GitHub Release](https://github.com/theihtisham/agent-shadow-brain/releases/tag/v5.0.1)

## Highlights

v5.0.1 makes the Shadow Brain **truly zero-config** — install it and everything works automatically. Auto-detect your project, auto-install MCP configurations, auto-connect to any AI tool, and auto-serve a rich web dashboard.

---

## New Features

### Zero-Config Auto-Setup
No manual configuration needed. On install:
- Detects project type (Node.js, Python, Go, Rust, Java, etc.)
- Detects installed AI tools (Claude Code, Cursor, Cline, etc.)
- Auto-generates MCP server configurations
- Auto-installs git hooks for continuous analysis
- Runs as postinstall hook — works on `npm install`

### MCP Server (19 Tools)
Full Model Context Protocol server compatible with:
- **Claude Code** — auto-installs to `.claude/mcp.json`
- **Cursor** — auto-installs to `.cursor/mcp.json`
- **Any MCP client** — standard protocol compliance

19 tools covering: health check, file analysis, pattern detection, security audit, memory recall, insight generation, and more.

### Rich Web Dashboard
Real-time WebSocket-powered dashboard with 8 panels:

| Panel | Content |
|---|---|
| Health | Overall score with trend arrow |
| Memory Tiers | Hot/warm/cold memory distribution |
| Insights | Live insight feed with priority coloring |
| Fixes | Smart fix suggestions with one-click apply |
| AI Tools | Connected agent status |
| Modules | Active brain module status |
| Stats | Performance metrics and counters |
| Controls | Start/stop/refresh brain controls |

**CLI:** `shadow-brain dashboard [--port 7890]`

### Natural Language Queries
Ask the brain anything in plain English:

```bash
shadow-brain ask "What are the most problematic files?"
shadow-brain ask "Show me security vulnerabilities"
shadow-brain ask "What patterns have you learned?"
```

### Plugin System
Extend the analysis pipeline with custom plugins:
- Hookable at any pipeline stage
- npm package discovery (`shadow-brain-plugin-*`)
- Plugin lifecycle management

### Brain Export/Import
Full brain state portability:

```bash
shadow-brain export brain-backup.json    # Export everything
shadow-brain import brain-backup.json    # Import on another machine
```

Perfect for team sync, backup, and migration.

---

## New CLI Commands

| Command | Description |
|---|---|
| `shadow-brain off` | Gracefully shut down all brain services |
| `shadow-brain ask <query>` | Natural language brain interaction |
| `shadow-brain export <file>` | Export brain state to JSON |
| `shadow-brain import <file>` | Import brain state from JSON |
| `shadow-brain plugin <action>` | Manage analysis plugins |
