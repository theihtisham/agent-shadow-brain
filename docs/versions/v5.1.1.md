## New Modules

### 1. Built-in LSP Server
Pure Node.js Language Server Protocol implementation — no external dependencies.

| Feature | Description |
|---|---|
| **Transport** | stdio (default) + TCP modes |
| **Diagnostics** | Real-time code analysis pushed to editor |
| **Hover** | Context-aware documentation on hover |
| **Code Actions** | Quick-fix suggestions integrated into editor |
| **Protocol** | Full LSP 3.17 message handling |

**CLI:** `shadow-brain lsp start [--port 7899]`

### 2. Custom LLM Fine-Tuning Engine
Trains on YOUR codebase patterns — generates training data automatically.

| Feature | Description |
|---|---|
| **Auto-Collection** | Records code changes as training pairs |
| **JSONL Export** | Compatible with OpenAI, Anthropic, local fine-tuning |
| **Multi-Model** | Supports training for multiple model targets |
| **Pattern Mining** | Extracts recurring patterns across changes |
| **Accuracy Tracking** | Measures suggestion quality over time |

**CLI:** `shadow-brain fine-tune train`, `shadow-brain fine-tune suggest`, `shadow-brain fine-tune stats`

### 3. Smart Cache (3-Tier Adaptive)
Multi-tier caching with predictive prefetch via co-access graph analysis.

| Tier | Capacity | Purpose |
|---|---|---|
| **Hot** | 200 entries | Sub-millisecond access, most-used |
| **Warm** | 1,000 entries | Frequently accessed, auto-promoted |
| **Cold** | 5,000 entries | Compressed, accessed on demand |

Features: LRU eviction with access-count weighting, tag-based invalidation, memoization (`getOrCompute`), predictive prefetch based on co-access patterns.

**CLI:** `shadow-brain cache stats`, `shadow-brain cache clear`

### 4. Intent Engine
Multi-strategy NLP for understanding developer intent from natural language.

| Strategy | Method | Best For |
|---|---|---|
| **Keyword** | Token matching | Direct commands |
| **Fuzzy** | Edit distance | Typos, variations |
| **Context-Aware** | Session history | Ambiguous commands |

Features: Work session tracking, action history, intent prediction, confidence scoring.

**CLI:** `shadow-brain intent predict <text>`, `shadow-brain intent stats`

### 5. Code DNA Fingerprinting
9-gene structural analysis that creates a unique "DNA profile" for any codebase.

| Gene Category | What It Measures |
|---|---|
| **Structure** | Function length, nesting depth, class complexity |
| **Complexity** | Cyclomatic complexity, cognitive complexity |
| **Style** | Naming conventions, formatting consistency |
| **Dependency** | Import patterns, coupling metrics |
| **Evolution** | Change frequency, age distribution |
| **Documentation** | Comment density, JSDoc coverage |
| **Error Handling** | Try/catch patterns, error propagation |
| **Testing** | Test coverage indicators, assertion patterns |
| **Performance** | Async patterns, loop complexity |

**CLI:** `shadow-brain dna profile [dir]`, `shadow-brain dna consistency [dir]`

### 6. Temporal Intelligence
Time-series analysis of code evolution with predictive capabilities.

| Feature | Description |
|---|---|
| **Velocity Metrics** | Daily/weekly/monthly activity, trend detection |
| **File Heatmaps** | Change frequency, hotness score, churn risk |
| **Bug Prediction** | 5-factor probabilistic model (bug rate, churn, recent changes, impact, staleness) |
| **Anomaly Detection** | Activity bursts, droughts, unusual hours, pattern breaks |
| **Peak Analysis** | Most productive hours and days of week |

**CLI:** `shadow-brain temporal velocity`, `shadow-brain temporal heatmap`, `shadow-brain temporal predict-bugs`
