# v5.2.0 — Agent Trust & Handoff Edition

> Released: 2026-04-24 | npm package: `@theihtisham/agent-shadow-brain@5.2.0`

## Highlights

v5.2.0 turns Shadow Brain from a memory layer into a safer agent coordination layer: explicit setup, agent handoff packets, firewall checks, proof timeline, and MCP auth controls.

**26 brain modules. 70+ CLI commands. 22 MCP tools. 10 agent adapters.**

---

## Trust Fixes

- CLI, dashboard, MCP server, portability exports, and version docs now report `5.2.0`.
- Install no longer silently modifies projects. Run `shadow-brain attach-all --dry-run` first, then attach explicitly.
- `shadow-brain detach` inspects installed artifacts directly instead of attaching before removal.
- MCP HTTP mode supports auth tokens and stricter origin handling.
- README and launch docs now describe the actual JSON-backed global brain at `~/.shadow-brain/global.json`.

---

## New Features

### Agent Safety Firewall

Checks commands, file paths, URLs, and content for destructive actions, prompt injection, secret exposure, and unsafe install patterns.

```bash
shadow-brain firewall check --command "rm -rf .env"
shadow-brain firewall hook
```

### Agent Handoff Packets

Generates a practical handoff from one coding agent to another with git state, changed files, recent memory, warnings, and next actions.

```bash
shadow-brain handoff codex claude --task "finish auth refactor"
```

### Proof Timeline

Shows recent cross-agent learning events and can generate a launch/share report.

```bash
shadow-brain timeline .
shadow-brain proof report . --output docs/launch/SHADOW_BRAIN_PROOF.md
```
